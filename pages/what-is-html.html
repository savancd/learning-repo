What is Html

Welcome to HTML!

HTML is the foundation of every web page. Every online page you've ever visited has HTML code, which your browser utilizes to display components such as text, pictures, and tables.

Photo by Jackson Sophat on Unsplash

!

HTML is the standard language for creating documents that will be viewed

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content.

Web browsers read HTML documents (containing HTML code) to display the resulting web page.

Text Paragraphs

HTML is based on tags. You can tell the web browser that you want to include a paragraph of text in your web page by enclosing the text in paragraph tags <p>example</p>.

Opening and Closing Tags

The paragraph element requires both opening and closing tags. <p> is the opening paragraph tag. It defines the start of the element. </p> is the closing tag, which is used to define the end of the element with the forward slash.

!

Many HTML elements require both opening and closing tags, also called container tags.
<p> I'm a paragraph </p>

HTML Elements

HTML uses tags to define elements. You‘ll keep learning about HTML elements in the following lessons. The HTML code example below creates a basic web page with 3 different HTML elements:

  1. A heading - defined using the <h> tag
  2. A paragraph - defined with the <p> tag
  3. An image - defined with the <img> tag
